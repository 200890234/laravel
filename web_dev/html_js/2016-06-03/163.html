jquery：jquery设置checkbox的选中状态，第二次打钩无效

                            

                                发布时间：2016-06-03   
                                发布者：mervyn   
                                
                                [ 信息报错 ]
                                [ 意见建议 ]

	原先使用jquery的1.7.3版本写checkbox切换选中和全选特效：




	$('#checkbox').attr('checked',true)

$('#checkbox').attr('checked',false)




	之后改用1.11.3版本后第一次全选没问题，但是第二次就不能打钩了，不过源码里还是执行的checked='checked'




	解决方案是：attr('checked',true)都换成prop('checked',true)




	prop()方法用于获取固有属性(href,class,id,src等)，attr用于获取用户自定义加上的属性
